AI-агент для похоронного бизнеса и наследства
Боль: Смерть — огромный стресс. Родственники не знают, какие аккаунты были у умершего, какие подписки надо отключить, где лежит криптокошелек или цифровое завещание.
Идея: «Digital Legacy Vault» (Цифровое хранилище наследия).

Суть: Приложение/сервис, где человек при жизни сохраняет цифровые ключи, пароли, письма близким, список активов. После смерти (или длительной неактивности) AI-агент аккуратно рассылает данные доверенным лицам.

Фишка: Автоматическое отключение подписок (Netflix, Spotify, облака) умершего, чтобы родственники не платили годами. Интеграция с госуслугами и нотариусами.

Монетизация: Подписка при жизни + разовый платеж за "архивацию" после смерти.
